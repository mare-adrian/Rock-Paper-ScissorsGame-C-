# Rock-Paper-ScissorsGame-C-
This is a simple console-based Rock–Paper–Scissors game written in C++. The player competes against the computer, and the first to reach 3 points wins the match.
✅ Features

User chooses between ROCK, PAPER, or SCISSORS

Computer selects a random option (1–3)

Score is tracked live for both player and computer

Displays round results: win, lose, or draw

Game ends when one player reaches 3 points

Input validation for incorrect words

Clean and easy-to-read C++ structure

✅ Technologies Used

C++

Basic control structures (if, switch)

Random number generation (rand(), srand())

Console input/output
